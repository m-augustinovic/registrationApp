# registrationApp
AngularJS SPA

#Getting started

#### 1. Install gulp:

```sh
$ npm install --global gulp-cli
```

```sh
$ npm install --save-dev gulp
```

#### 2. Install dependencies:

```sh
$ npm install
```

#### 3. Run with gulp:

```sh
$ gulp
```


For getting the storage view change URL path to http://localhost:8000/#/storage
